# stoke

A multi-language build tool with automatic venv management and reproducible builds.

> Currently supports Python. Java and C/C++ support is planned.

## Installation

```bash
pip install stoke-build
```

Requires Python 3.11 or higher.

## Quick Start

```bash
mkdir myapp
cd myapp
stoke init
stoke build
```
